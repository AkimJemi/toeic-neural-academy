# TOEICプロジェクト - ワークフロー

このディレクトリには、TOEICプロジェクトの定型作業手順を記述したワークフローファイルを配置します。

## ワークフロー作成ガイド

各ワークフローファイルは以下の形式で作成してください：

```markdown
---
description: [ワークフローの簡潔な説明]
---

## 目的
[このワークフローの目的]

## 前提条件
- [必要な環境・ツール]

## 手順
1. [ステップ1]
2. [ステップ2]
...

## 注意事項
- [重要な注意点]
```

## 推奨ワークフロー

- `deploy.md` - デプロイ手順
- `test.md` - テスト実行手順
- `database_migration.md` - DBマイグレーション手順
- `bug_fix.md` - バグ修正フロー
- `feature_development.md` - 新機能開発フロー
- `tts_testing.md` - TTS機能テスト手順
- `score_calibration.md` - スコア予測キャリブレーション手順
